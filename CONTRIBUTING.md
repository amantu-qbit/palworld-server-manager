# Contributing to Palworld Server Manager

Thanks for your interest in improving Palworld Server Manager! Contributions of all kinds — bug reports, features, docs, and fixes — are welcome.

## Getting set up

```bash
# Install dependencies
npm install

# Run the app in development mode
npm run tauri dev
```

See the [README](README.md#build-from-source) for the full list of prerequisites (Node.js 20+, Rust, Microsoft C++ Build Tools, WebView2).

## Before you open a pull request

- Run the test suite: `npm test`
- Make sure the app builds: `npm run build`
- Keep TypeScript **strict** — no new type errors — and follow the existing code style.
- Write clear, descriptive commit messages that explain the "why" of your change.

## Code of conduct

Please be respectful and constructive in issues, pull requests, and discussions. We want this to be a welcoming project for everyone.

Happy hacking!
